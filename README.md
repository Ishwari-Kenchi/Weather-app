
# 🌦 Weather App

A modern and interactive **Weather Application** built using **HTML, CSS, and Vanilla JavaScript** that fetches real-time weather data using the **OpenWeatherMap API**.

The app allows users to search for any city worldwide and view accurate weather information instantly.

---

## 🚀 Features

- 🌍 Search weather for **any city in the world**
- 🌡 Displays **temperature in Celsius**
- 🔄 **Temperature toggle** to convert between **Celsius (°C) and Fahrenheit (°F)**
- 💧 Shows **humidity**
- 💨 Displays **wind speed**
- 🤒 Shows **feels like temperature**
- ⛅ Dynamic **weather icons**
- 🎨 **Dynamic background colors based on weather conditions**
- ⚠️ Error handling for invalid city names
- 📱 Fully **responsive design**

---

## 🎨 Dynamic Weather Background

The background color changes automatically based on the weather condition:

- ☀️ **Sunny / Clear weather** → Yellow / Orange gradient
- 🌧 **Rainy weather** → Blue / Dark gradient
- ☁️ **Cloudy weather** → Grey tones
- ❄️ **Snow weather** → Light cool colors

This makes the app visually interactive and improves the user experience.
